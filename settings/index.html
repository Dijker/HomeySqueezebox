HomeySqueezebox

Welcome to the HomeySqueezebox app. You should enter the settings for your Squeezebox system here before attempting to add any Squeezebox devices.

-- THIS APP IS ALPHA: Please see notes below --

Please be aware that this app is in alpha. Hopefully, everything that is present should work but you might expect some hiccups along the way. If you spot anything, please report it on the GitHub page or in the Athom forums. You can also expect significantly more features to be added over time.

Please note that the plugin will not operate correctly until you have manually specified a Logitech Media Server address. Automatic discovery may be added at a later date.

Logitech Media Server

Please specify the settings for your Logitech Media Server below. You should include the "http://" part and can use a fully qualified domain, a local name, or an IP address. In most cases, leaving the Server Port at the default 9000 is the correct choice.
